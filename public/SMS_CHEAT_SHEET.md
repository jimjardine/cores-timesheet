# Daily Text — Cheat Sheet

Text this to **+1 (506) 404-6969** (or WhatsApp **+1 (415) 523-8886** — first time, send `join cookies-could` once before your first real message) every work day:

> In 7:30, 4760 6hrs rebuilt port engine bearings, 4862 2hrs fuel lines, out 4, lunch 30, no PD

## Include these 5 things

1. **Start time** — "In 7:30"
2. **Each job** — job number + hours + what you did: "4760 6hrs rebuilt bearings"
3. **Finish time** — "Out 4:30"
4. **Lunch** — "lunch 30" or "no lunch"
5. **Per diem** — where you're staying, or "no PD"

Used supplies? Add them too: "supplies brake cleaner x1, wire brushes x2"

## Copy-paste template

Don't want to type it from scratch every day? Text **TEMPLATE** to either number and it'll text this back to you — or just copy it from here, fill in the blanks, delete any lines you don't need, and send it:

```
In: [time]
Job [job#]: [hours]hrs - [task]
Job [job#]: [hours]hrs - [task]
Out: [time]
Lunch: [minutes]
PD: [y/n]
Supplies: [product] x[qty]
```

Add more `Job` lines if you worked more than two. Double-tap a `[bracketed]` word to select and replace it in one go — no spaces inside the brackets means you never have to drag-select across words. First time using this? Send a filled-in test message and make sure you get a "Got it" reply back before relying on it for a real day.

## Missing a job number?

Text **JOBS** for the full list, or **JOBS + boat name** for just that boat.

## Want to check what you've logged?

Text **TIMESHEET** (or **TS**) — add a day like "TS yesterday" for another date.
